По текущему состоянию репозитория:

Централизованной темы/констант цвета нет — цвета захардкожены в StyleSheet.create по экранам/компонентам.

Следствие: шаг 1 редизайна — это не find-replace, а введение theme-слоя (src/theme/theme.js + ThemeProvider/useTheme) и консолидация:

все «фиолетовые» (#6B4EFF/#6C63FF/#6B46C1/#8B5CF6) → токен accent;
инлайн-нейтрали → t1…t4 / surface / hairline;
градиенты (ProgressBar) → accent + accentPress либо bgGradient.

Эталонные значения токенов — в приложенных theme.js / ThemeProvider.js (можно взять как есть или адаптировать в архитекторском чате).
