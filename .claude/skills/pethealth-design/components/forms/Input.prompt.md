Text input on a solid surface; border lifts to mint accent on focus with a soft ring.

```jsx
<Input icon="mail-outline" placeholder="Email" />
<Input icon="search-outline" placeholder="Ask the assistant…" pill />
```

- `icon` takes an Ionicons name (tints to accent on focus).
- `pill` for chat/search style; default is the 12px rounded field used in forms.
